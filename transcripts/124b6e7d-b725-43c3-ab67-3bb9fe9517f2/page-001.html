## Quick Start

```bash
nix run github:sadjow/claude-code-nix
```

**Install to your system:**
```bash
nix profile install github:sadjow/claude-code-nix
```

## Runtime Options

| Runtime | Package | Binary | Command |
|---------|---------|--------|---------|
| Native (default) | `claude-code` | `claude` | `nix run .#claude-code` |
| Node.js 22 LTS | `claude-code-node` | `claude-node` | `nix run .#claude-code-node` |
| Bun | `claude-code-bun` | `claude-bun` | `nix run .#claude-code-bun` |

## Using as Flake Input in devShell

Add to your `flake.nix`:

```nix
{
  inputs = {
    nixpkgs.url = "github:NixOS/nixpkgs/nixpkgs-unstable";
    claude-code.url = "github:sadjow/claude-code-nix";
  };

  outputs = { self, nixpkgs, claude-code }: {
    devShells.default = nixpkgs.mkShell {
      inputsFrom = [ claude-code.devShells.default ];
      buildInputs = with nixpkgs; [
        claude-code.packages.${system}.claude-code
        # ... other dev dependencies
      ];
    };
  };
}
```

## MiniZinc Solver Configuration Files (.msc)

A solver configuration file must have the file extension .msc (for MiniZinc Solver Configuration).

## .msc File Locations

Configuration files must be placed in the minizinc/solvers/ directory of the MiniZinc installation (which can be found at /usr/share/minizinc/solvers on Linux systems, inside the MiniZincIDE application on macOS, and in the Program Files\MiniZinc IDE (bundled) folder on Windows).

Additional locations include:

- The directory $HOME/.minizinc/solvers on Linux and macOS systems, and the Application Data directory on Windows systems
- Any directory listed on the MZN_SOLVER_PATH environment variable (directories are separated by : on Linux and macOS, and by ; on Windows systems)
- Any directory listed in the mzn_solver_path option of the global or user-specific configuration file

## MZN_SOLVER_PATH Environment Variable

You can install solvers in places other than the standard PREFIX for the system by extending the MZN_SOLVER_PATH environmental variable with "LOCAL_PREFIX/share/minizinc/solvers".

## .msc File Content

The configuration file is a simple file in JSON or .dzn format that contains basic information such as the solver's name, version, where its library of global constraints can be found, and a path to its executable.

Key fields include:
- executable (required): The executable for the solver that can run FlatZinc files, which can be just a file name, an absolute path, or a relative path (interpreted relative to the location of the configuration file)
- mznlib: The solver-specific library of global constraints and redefinitions, which should be a directory name (either absolute or relative to the configuration file location)

## Selecting a Solver

You can use --solver <id> or --solver <solver configuration file>.msc to select which solver to use.
