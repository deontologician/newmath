3 edit · 2 fetch · 1 search · 1 bash

Done. Added 6 new properties from the Magma (algebra) Wikipedia article:

  • Idempotent: a·a = a
  • Medial: (a·b)·(c·d) = (a·c)·(b·d)
  • Left distributive: a·(b·c) = (a·b)·(a·c)
  • Right distributive: (b·c)·a = (b·a)·(c·a)
  • Left cancellative: a·b = a·c → b = c (every row is a permutation)
  • Right cancellative: b·a = c·a → b = c (every column is a permutation)

Both cancellative together make it a Latin square (quasigroup).
