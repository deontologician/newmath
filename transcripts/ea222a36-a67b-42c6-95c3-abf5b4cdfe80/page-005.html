## Magma Definition and Properties

A magma is a set with a binary operation. Various algebraic properties can be defined for magmas:

### Key Properties

**Medial:** A magma satisfies the identity xy • uz ≡ xu • yz, or more explicitly, (x • y) • (u • v) = (x • u) • (y • v).

**Idempotent:** A magma satisfies the identity xx ≡ x.

**Unipotent:** A magma satisfies the identity xx ≡ yy.

**Autodistributive:** A magma is both left and right distributive, where left distributive means it satisfies the identity x • yz ≡ xy • xz and right distributive means it satisfies the identity yz • x ≡ yx • zx.

### Related Properties

The search results also mention several related properties:
- Left semimedial: xx • yz ≡ xy • xz
- Right semimedial: yz • xx ≡ yx • zx
- Commutative: xy ≡ yx
- Zeropotent: xx • y ≡ xx ≡ y • xx

### Connections Between Properties

Any commutative semigroup is a medial magma, and a medial magma has an identity element if and only if it is a commutative monoid.
