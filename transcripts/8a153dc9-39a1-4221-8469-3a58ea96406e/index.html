Key data types

Expression tree (src/expr/tree.ts)

type Expr = SymbolNode | OpNode
interface SymbolNode { kind: 'symbol'; value: number }  // 0-indexed
interface OpNode { kind: 'op'; left: Expr; right: Expr }
  • A redex = OpNode where both children are SymbolNodes (innermost binary op)
  • reduce(expr, targetNode, table) finds target by reference identity (===), returns new tree with redex replaced by result symbol
  • randomExpr(symbolCount, leafCount) builds a random tree by combining adjacent leaves — 4–6 leaves gives 3–5 reduction steps

Operation table (src/table/types.ts)

interface OpTable { size: number; cells: number[][] }  // 0-indexed
  • randomTable(3) for initial version (no solver needed)
  • fromSolverResult(op) converts solver's 1-indexed output to 0-indexed

Rendering approach

  • Expression: Each tree node → a <span>. Redex groups get class="redex-group" + tabindex="0" + click handler. A WeakMap<HTMLElement, OpNode> maps DOM elements to tree nodes.
  • Table: Standard <table> with id="cell-{row}-{col}" on each td for animation targeting. Each cell gets data-value for CSS color rules.
  • After each reduction: Full re-render of the expression from the updated tree (simple, correct, negligible cost for ~10 nodes).

Animation sequence (src/expr/animate.ts)

When user clicks a redex (A ○ B):

  1. Flash (200ms): Cell [A][B] in table gets .cell-flash — scale up + glow
  2. Fly (400ms): Clone result symbol at cell position, Web Animations API translates it to redex position (position: fixed)
  3. Collapse (300ms): Redex-group animates max-width → 0, opacity → 0. Result symbol inserted adjacent, fades in.
  4. Re-space (200ms): Flex layout naturally adjusts. Remove old DOM, update tree, full re-render.

Lock flag (state.animating) prevents clicks during animation.

Visual design

  • Background: true black #000
  • Font: Cormorant Garamond (Google Fonts) — elegant serif with math feel
  • Symbol colors: X = #ff6b6b (red), Y = #4ecdc4 (teal), Z = #ffe66d (gold)
  • Layout: CSS Grid — table left, equation right on wide screens; stacked on mobile
  • Redex hover: subtle purple glow indicating clickability
  • Completion: final symbol pulses with a glow animation
